Stock Market Data Analysis 

Overview:
Academic data analysis project using Python to explore a stock market dataset with return indicators.

Dataset:
- File: `data/data_with_indicators1.csv`
- Format: CSV
- Educational dataset used for analysis and visualization.

 What I did:
- Checked missing values and filled selected return columns using median
- Converted `Date` and extracted `Year`, `Month`, `Day`
- Built a summary table by date
- Visualized yearly record counts and return indicators over time

Tools:
Python, Pandas, NumPy, Seaborn, Matplotlib, Jupyter Notebook.
